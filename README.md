Hangman-Game
====
A simple word-guessing game called Hangman in java.
This is a computer versus user version. User does not know whether he/she plays traditional version or evil version.

Traditional Hangman
--------
The computer will pick a random word from the dictionary.


Evil Hangman
--------
The computer will always change the target word during the guessing.
